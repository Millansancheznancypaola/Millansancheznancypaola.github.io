Consulta #6.

Obtener listado de profesores y asignaturas que imparten

SELECT profesor.nombre, profesor.apellido1, profesor.apellido2,asignatura.nombre
FROM profesor
JOIN impartir
ON profesor.idProfesor = impartir.idProfesor
JOIN asignatura ON impartir.idAsignatura = asignatura.idAsignatura
ORDER BY profesor.idProfesor, asignatura.idAsignatura;

Consulta #7.

Mostrar la asignatura que tiene mas alumnos inscritos y su total

SELECT asignatura.nombre,
COUNT(matricula.idAlumno) AS inscripciones FROM asignatura
JOIN matricula ON asignatura.idAsignatura = matricula.idAsignatura
GROUP BY asignatura.nombre ORDER BY `inscripciones` DESC

Consulta #8.

Obtener listado de alumnos que les da clase con ID PR048(JOSE MANUEL),sin importar en que asignatura esté inscrito.

SELECT alumno.idAlumno, alumno.nombre, alumno.apellido1, alumno.apellido2, alumno.email
FROM alumno JOIN matricula ON alumno.idAlumno = matricula.idAlumno
JOIN impartir ON matricula.idAsignatura = impartir.idAsignatura

Consulta #9.

Obtener listado de cursos, con asignatura, y el maestro que las imparte

SELECT curso.idCurso, curso.nombreDescriptivo
AS nombre_curso, asignatura.idAsignatura,
asignatura.nombre AS nombre_asignatura,
profesor.idProfesor, profesor.nombre AS nombre_profesor,
profesor.apellido1 AS apellido1_profesor,
profesor.apellido2 AS apellido2_profesor
FROM curso JOIN asignatura ON curso.idCurso = asignatura.curso
JOIN impartir ON asignatura.idAsignatura = impartir.idAsignatura JOIN profesor ON impartir.idProfesor = profesor.idProfesor
ORDER BY curso.idCurso, asignatura.idAsignatura, profesor.idProfesor;

Consulta #10.

Obtener listado de alumnos, con asignaturas que cursa, profesor que las imparte yal curso al que pertenece

SELECT alumno.idAlumno,alumno.nombre AS nombre_alumno, alumno.apellido1
AS apellido1_alumno,alumno.apellido2
AS apellido2_alumno,
asignatura.idAsignatura,asignatura.nombre
AS nombre_asignatura,profesor.idProfesor,
profesor.nombre AS nombre_profesor,profesor.apellido1
AS apellido1_profesor, profesor.apellido2 AS apellido2_profesor, curso.idCurso,
curso.nombreDescriptivo AS nombre_curso FROM alumno
JOIN matricula ON alumno.idAlumno = matricula.idAlumno
JOIN asignatura ON matricula.idAsignatura = asignatura.idAsignatura
JOIN impartir ON asignatura.idAsignatura = impartir.idAsignatura
JOIN profesor ON impartir.idProfesor = profesor.idProfesor
JOIN curso ON asignatura.curso = curso.idCurso
ORDER BY alumno.idAlumno, asignatura.idAsignatura;
